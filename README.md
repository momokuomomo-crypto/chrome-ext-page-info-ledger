# chrome-ext-page-info-ledger（ページ情報ワンクリック台帳）

閲覧中のページのタイトル・URL・選択テキストを、ワンクリックで手元の台帳（`chrome.storage.local`）に記録するChrome拡張機能（Manifest V3）。

## 使い方

1. ツールバーのアイコンをクリックするとポップアップが開く
2. 「現在のページを追加」ボタンで、そのページのタイトル・URL・選択中のテキストを記録する
3. ページ上でテキストを選択してから右クリック →「選択範囲を台帳に追加」でも記録できる
4. ポップアップの「台帳を開く」（または右クリックメニューの「台帳を開く」）で、記録した一覧・CSV/JSONエクスポート・全件削除ができる専用ページを開く
5. 直前の記録を間違えた場合は、ポップアップの「直前の追加を取り消す」（または右クリックメニュー）で取り消せる

同一URL・同一選択テキストを10秒以内に連続で記録しようとした場合は重複として無視される。

## セットアップ

```bash
npm install
npm run build
```

`chrome://extensions` でデベロッパーモードを有効にし、「パッケージ化されていない拡張機能を読み込む」で `dist/` を選択する。

## 権限

| 権限 | 用途 |
| --- | --- |
| `activeTab` | クリック操作の起点となったタブのURL・タイトル取得、選択テキスト取得 |
| `scripting` | 選択テキストを取得するスクリプトの実行 |
| `storage` | 記録した台帳データの保存（`chrome.storage.local`） |
| `contextMenus` | 右クリックメニュー（選択範囲の追加・台帳を開く・取り消し） |
| `downloads` | 台帳ページからのCSV/JSONエクスポート |

http(s)以外のページ（`chrome://`等）は対象外。
